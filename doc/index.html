Core Features

📁

File Management

Intuitive file browsing interface with tree view, grid view, and list view. Support for file operations including copy, move, and safe delete with recycle bin mechanism.

🏷️

Tag System

Powerful tag management system supporting custom tags, multi-tag combinations, and tag statistics. Organize files by theme, time, location, or status.

🔍

Advanced Search

Multi-condition combination search supporting filename, tags, file type, date range, and file size filters. Real-time search results with sorting options.

🔗

Similarity Detection

Intelligent duplicate file detection based on content analysis. Automatic grouping, similarity sorting, and side-by-side preview comparison.

📊

Metadata Management

Automatic EXIF information extraction including shooting time, GPS location, camera model, and shooting parameters for comprehensive file analysis.

⚡

Real-time Updates

Server-Sent Events (SSE) technology for real-time file list updates, task progress notifications, and system status synchronization.
